# GRUNDREGELN

## Spirale
- im Kreis häkeln  
- keine Luft- oder Kettmasche  
- Maschenmarker immer in die **1. Masche der Runde**

## Reihen
- am Reihenende **1 Luftmasche**, wenden  
- Luftmasche zählt **nicht** als Masche

## Abkürzungen

| Abkürzung | Bedeutung |
|-----------|-----------|
| fM | feste Masche |
| BLO | hinteres Maschenglied |
| FLO | vorderes Maschenglied |
